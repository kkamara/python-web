# python-web



## Requirements

* [Python](https://www.python.org).

## Installation

```bash
pip3 install virtualenv
virtualenv .venv
source .venv/bin/activate
(.venv) pip3 install -r requirements.txt
```

## Usage

```bash
(.venv) # alias py3="python3"
(.venv) py3 manage.py runserver 8000
```
